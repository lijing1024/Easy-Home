# Easy-Home
一个基于React全家桶的移动端租房项目
项目业务概述

- 登录
- 主页
  - 房源列表
  - 利率计算器
  - 地图找房
- 资讯
  - 头条与资讯
  - 问答
- 微聊
  - 聊天列表
  - 聊天窗口
  - 聊天消息
- 个人中心
  - 个人信息布局
  - 头像上传与裁切

开发模式和技术选型

- 开发模式：前后端分离
- 技术选型：React全家桶
  - React
  - React-router
  - axios
  - 官方脚手架create-react-app
  - 第三方UI组件库SemanticUIReact
  - 轮播图组件
  - 图表组件
  - 地图API
